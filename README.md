# Inegbedion Master Thesis Documentation

## Requirements
- Python 3.8 or higher
- Required libraries: pandas, numpy, scikit-learn, matplotlib

## Architecture
- The project is structured into the following main components:
  - `data/`: Contains the raw data and processed datasets.
  - `notebooks/`: Jupyter notebooks for data exploration and analysis.
  - `src/`: Source code for data processing and model training.

## Raw Data Location
- The raw data for this project can be found on Zenodo. You can access it [here](https://zenodo.org/record/[your Zenodo record id]). Replace `[your Zenodo record id]` with the correct Zenodo record.

## Fun Folder
- The `fun/` folder contains additional resources, including scripts for data visualization and exploratory data analysis. These scripts assist in better understanding the dataset and validating the results obtained from the models.
